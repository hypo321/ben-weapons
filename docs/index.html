The Three Weapons

Forbidden Sword

A netherite sword imbued with blaze energy. Type /sword ability activate while holding it to launch a powerful fireball in the direction you're looking.

🔥 /sword ability activate ⏳ 30s cooldown 🛡 Fire Resistance (passive)

Forbidden Axe

A gold-tipped axe crackling with storm energy. Automatically calls down a lightning bolt on your target every time you land a hit — no button press needed.

⚡ Auto-strikes on hit ⏳ 25s cooldown 💚 Regen II for 20s (on proc)

Forbidden Mace

A mace bound with breeze and echo energy. Press F (swap-hands) to launch yourself ~30 blocks forward. Comes pre-enchanted with Density V and Breach IV.

💨 F key to dash ⏳ 15s cooldown 🏃 Speed II (passive) ⚒ Density V + Breach IV

Installation

1

Download the plugin

Click the Download BenWeapons.jar button above — it downloads the latest version directly, no extra clicks needed.

⚠️ If you've downloaded it before, your browser may save it as BenWeapons(1).jar. Make sure the file you upload is named exactly BenWeapons.jar — rename it if needed before uploading.

2

Replace the old file

Go to your server's plugins/ folder. Delete any existing BenWeapons.jar first, then upload the new one. If you skip the delete step, your hosting panel may keep both files.

3

Fully restart the server

Stop and start the server — do not use /reload. A reload will not pick up the new JAR. You must do a full stop → start cycle.

4

Confirm the version

Once back in-game, run /benweapons version. It will show the version number — check it matches the release you just downloaded before reporting any issues.

5

Craft away!

All three weapons appear in the in-game recipe book automatically. No additional configuration required.

Requirements: Paper 1.21.4+  ·  Java 21+

Custom Weapon Textures (Resource Pack)

The resource pack gives each weapon its own unique look. Players who decline it will still see normal netherite sword / axe / mace visuals — no gameplay impact.

✨ No setup required. The plugin handles everything automatically. Just install the JAR as normal — the resource pack is sent to every player when they join, with no server.properties changes needed.
1

Install the JAR as normal

The resource pack is bundled into BenWeapons.jar. No separate download or configuration needed.

2

Players are prompted on join

When a player joins, the server automatically offers them the resource pack. If they accept, all three weapons display Ben's custom textures immediately.

Players who decline can still play normally — weapons will just look like standard netherite items.

3

Updates are automatic too

When you install a new version of the JAR, the updated pack is pushed to players automatically on their next join — no manual SHA-1 or URL changes required.

Already have a resource pack on your server? The plugin's pack is sent in addition to any pack already set in server.properties — they won't conflict.

Commands

All players can use the ability and trust commands. The give command requires OP.

/sword ability activate

Activate the Forbidden Sword — must be holding it in your main hand.

/trust <player>

Your weapons will no longer harm this player. Both players receive a notification. Trust persists across restarts.

/untrust <player>

Revoke trust — your weapons can harm them again.

/benweapons <sword|axe|mace> [player]

Give a custom weapon to yourself or another player. Requires OP (or benweapons.give permission). Omit [player] to give to yourself.
